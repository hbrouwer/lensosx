What is Lens(OSX)?

Lens is the light, efficient network simulator, written by Doug Rohde. LensOSX is a native MacOSX port of Lens that runs on MacOSX 10.5 or higher, created by Harm Brouwer and Hartmut Fitz.

Disclaimer: LensOSX is ported "as is". This means that any bugs present in the original Lens source code might also be present in LensOSX.

Running LensOSX

You can start LensOSX from the applications folder, or through spotlight. LensOSX will run with a graphical interface and its own console [screenshot 1].

Or, you can start LensOSX from a terminal. It will still run with a graphical interface, but now print its console output to the terminal [screenshot 2]:

$ /Applications/LensOSX.app/Contents/MacOS/LensOSX

Finally, you can also run LensOSX in batch mode [screenshot 3]:

$ /Applications/LensOSX.app/Contents/MacOS/LensOSX -b

I found a Bug, what now?
