  This command will bring up the Lens manual in a web browser. The browser
  executable is taken from the WEBBROWSER environment variable. If that is
  not defined, the default is "netscape".

  If Netscape is being used and a window is already open on the current
  display, the manual will be opened in that window. Otherwise a new browser
  will be spawned.
